# Kassim Idris

Kassim Idris presents a curated collection of essays and poetry, exploring themes of identity, memory, and the landscapes—both urban and natural—that shape experience. This collection is designed to offer a tactile, layered experience, reflecting warmth, contrast, and subtle textures.
